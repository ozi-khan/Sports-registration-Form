# Features:

- Participant registration form

- Data submission using POST method

- Stores participant data in SQLite database

- Displays registered participants in a table

- Uses Jinja2 templates

- Clean project structure

- Basic validation

- Flash messages for user feedback

- Implementing sessions for login system

  ## Tech Stack:

- Python

- Flask

- Flask-SQLAlchemy

- SQLite

- HTML / CSS

- Flask-sessions
